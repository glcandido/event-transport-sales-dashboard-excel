# Event Transport Sales Analytics Dashboard

This project is a public, anonymised Excel dashboard built from a sales reporting workflow for an event transport business.

The dashboard summarises sales activity across months, customer type and anonymised event services. It was designed to turn raw order-level data into an interactive business-facing report with KPIs, charts, slicers and portfolio-safe data anonymisation.

## Project Objective

The goal was to create a dashboard that helps users understand:

- Overall sales performance
- Monthly sales and order trends
- Average order value and items per order
- New vs returning customer contribution
- Top-performing anonymised event services
- Month-over-month performance changes

## Tools Used

- Microsoft Excel
- Power Query
- PivotTables
- PivotCharts
- Slicers
- KPI cards
- Data anonymisation

## Key Features

- Executive KPI cards for revenue, orders, AOV, items sold and revenue per item
- Monthly trend analysis
- Top event/service ranking
- New vs returning customer revenue and order split
- Month-over-month KPI comparison
- Interactive slicers for month, event and customer type
- Public-safe event labels such as `Event 001`, `Event 002`
- Currency transformed and displayed with generic `$` formatting
- Dates reduced to month-level reporting

## Data Privacy

This workbook uses a public-safe version of the original analysis.

To protect confidentiality:

- Company name removed
- Event, artist, venue and route names anonymised
- Pickup/location details removed
- Dates reduced to month/year level
- Revenue values transformed
- No customer-level personal data included

## What I Built

1. Imported and cleaned order-level sales data

3. Created PivotTables and PivotCharts for reporting
4. Added slicers for interactive filtering
5. Built KPI cards and derived metrics
6. Added month-over-month performance logic
7. Created a public-safe version for portfolio use
